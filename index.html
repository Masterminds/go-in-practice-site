Go in Practice is a cookbook style book of patterns for Go development. This book contains patterns for DevOps, CloudOps, building cloud and web services, leveraging the cloud, web development, and much much more.

The Manning Early Access Program (MEAP) allows you to have access to the book while it is being written. During this time you can read the book and provide feedback along the way. You can learn more on the Manning Publication website.


About Go

Go, sometimes referred to as golang, is a modern systems programming language. Go provides the tools to rapidly build web, cloud, and systems applications.

Designed with modern hardware in mind, Go is a statically typed and compiled language providing garbage collection, concurrency, multiple return values, a modern standard library, package management, and more. Go is an engineered language designed to solve real world problems. It was born from the experience of luminary software engineers.

Techniques

Most of the chapters contain topic introductions followed by a series of recipe like techniques. Each technique shares a recipe for solving a problem with a discussion on the problem and solution.

The book contains techniques for:

  • Handling Linux style flags in your application.
  • CLI commands and sub-commands.
  • How to handle URL routing.
  • Multiple ways to handle application configuration.
  • Patterns for working with HTML Templates.
  • Serving static assets.
  • Working with REST APIs (including version handling).
  • Communicating between cloud services.
  • Microservices and High Availability.
  • Working with Databases.
  • Utilizing Infrastructure as a Service (IaaS).
  • Running in a Platform as a Service (PaaS).
  • And many more...

Sample Chapter
